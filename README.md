# ふわっち 自動録画アプリ

ふわっち（whowatch.tv）の配信を自動検知・録画・コメント保存するWindowsアプリです。

## 機能

- 配信者を登録しておくと配信開始を自動検知して録画開始
- 複数配信者の同時録画
- コメント・アイテムのリアルタイム表示・保存
- 配信ごとのコメント切り替えタブ
- ブラウザUIで操作（http://localhost:5000）
- 字幕埋め込み（コメントをMP4に焼き込み）
- リスナー名クリックでプロフページへ

## 必要なもの

- Windows 10/11
- Python 3.10以上（https://python.org）
- ffmpeg.exe（https://ffmpeg.org → ffmpeg.exeをstart.batと同じフォルダに置く）

## インストール

1. このリポジトリをダウンロード（ZIP展開 または git clone）
2. ffmpeg.exeを同じフォルダに置く
3. `start.bat` をダブルクリック
4. ブラウザが自動で開く（http://localhost:5000）

## 使い方

1. 左サイドバーに配信者のuser_path（例: `w:Nanoha_323`）を入力して追加
2. 「▶ 自動監視 開始」をクリック
3. 配信が始まると自動録画スタート

### user_pathの調べ方

ふわっちのプロフページURL `https://whowatch.tv/profile/w:XXXXX` の `w:XXXXX` 部分

## 設定

| 項目 | 説明 |
|------|------|
| チェック間隔 | 配信開始をチェックする間隔（秒） |
| 録画画質 | 高画質推奨 |
| 保存フォルダ | 録画ファイルの保存先 |
| コメント保存 | ONでコメントをtxtに保存 |
| 字幕埋め込み | ONで録画終了後にMP4に字幕を焼き込み |

## 保存ファイル

```
保存フォルダ/
  配信者名_タイトル_日時.ts      # 録画動画
  配信者名_タイトル_日時.txt     # コメントログ
  配信者名_タイトル_日時_sub.mp4 # 字幕埋め込み動画（設定ONの場合）
```

## アプリの終了

`stop.bat` をダブルクリック、またはタスクマネージャーでpython.exeを終了
